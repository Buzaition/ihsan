# إحسان — Holy Quran & Azkar Web App

**إحسان** هو موقع قرآني عربي بسيط وهادئ لقراءة القرآن الكريم، البحث في الآيات، قراءة الأذكار، متابعة مواقيت الصلاة، استخدام المسبحة الإلكترونية، وحفظ المفضلة وموضع القراءة.

تم تطوير التطبيق كعمل نافع وإهداء إلى روح الجدة:

> **إحسان إبراهيم الخبيري — رحمها الله وغفر لها وأسكنها فسيح جناته.**

---

## فكرة المشروع

المشروع عبارة عن تطبيق ويب Frontend فقط يعمل بملفات:

- `index.html`
- `styles.css`
- `app.js`
- `manifest.webmanifest`
- مجلد `assets`

ولا يحتاج إلى Backend أو قاعدة بيانات خارجية. يتم حفظ بيانات المستخدم مثل آخر موضع قراءة، المفضلة، الإعدادات، والمسبحة باستخدام `localStorage` داخل المتصفح.

---

## المميزات

### القرآن الكريم

- فهرس كامل لسور القرآن الكريم.
- قراءة السور آية آية في بطاقات منفصلة.
- وضع قراءة متصلة للسورة كاملة.
- عرض البسملة بشكل صحيح:
  - تظهر مستقلة في منتصف القراءة في كل السور.
  - لا تظهر مستقلة في سورة الفاتحة لأن الآية الأولى هي البسملة.
  - لا تظهر في سورة التوبة.
- أرقام آيات مزخرفة في وضع القراءة المتصلة.
- السورة السابقة والسورة التالية.
- حفظ موضع القراءة تلقائيًا.
- شاشة كاملة للقراءة.
- عرض التفسير أسفل الآية في القراءة المتصلة.
- تشغيل صوت الآية.
- منع تداخل الأصوات عند تشغيل أكثر من آية.
- القارئ الافتراضي: **الحصري**.

### البحث

- بحث في القرآن الكريم.
- البحث غير حساس للتشكيل والهمزات والمدود وعلامات الوقف.
- عرض النتائج بشكل واضح مع اسم السورة ورقم الآية.

### الأذكار

- قسم كامل للأذكار.
- عرض أبواب الأذكار في بطاقات منظمة.
- عند اختيار باب، تظهر الأذكار أسفل نفس مكان الاختيار.
- عداد تكرار لكل ذكر.
- نسخ ومشاركة وحفظ الأذكار في المفضلة.

### المسبحة الإلكترونية

- عداد تسبيح كبير وسهل الاستخدام.
- اختيار ذكر سريع.
- تصفير العداد.
- حفظ الذكر في المفضلة.

### مواقيت الصلاة

- عرض الساعة الحالية.
- عرض تاريخ اليوم.
- عرض التاريخ الهجري.
- عرض مواقيت الصلاة.
- عرض الصلاة القادمة.
- عرض الوقت المتبقي على الصلاة القادمة.
- دعم استخدام الموقع الجغرافي عند موافقة المستخدم.

### المفضلة والمتابعة

- حفظ الآيات والأذكار.
- فتح موضع الآية من المفضلة.
- حذف عنصر من المفضلة أو حذف الكل.
- متابعة ختمة بسيطة.
- حفظ آخر موضع قراءة تلقائيًا.

### التصميم وتجربة الاستخدام

- واجهة عربية RTL.
- تصميم هادئ مناسب لمحتوى القرآن والأذكار.
- وضع ليلي / نهاري.
- متجاوب مع الهاتف، التابلت، والكمبيوتر.
- أيقونة مخصصة للتطبيق.
- مناسب للنشر على GitHub و Vercel.

## التشغيل محليًا

يفضل تشغيل المشروع من خلال سيرفر محلي، وليس بفتح ملف `index.html` مباشرة، لأن بعض المتصفحات تمنع طلبات البيانات عند التشغيل من `file://`.

### الطريقة الأولى باستخدام Python

افتح Terminal داخل مجلد المشروع، ثم شغّل:

```bash
python -m http.server 5500
```

ثم افتح في المتصفح:

```text
http://localhost:5500
```

### الطريقة الثانية باستخدام VS Code

يمكنك استخدام إضافة **Live Server** ثم تشغيل المشروع من ملف `index.html`.

## النشر على GitHub

1. أنشئ Repository جديد على GitHub.
2. ارفع ملفات المشروع في جذر المستودع:
   - `index.html`
   - `styles.css`
   - `app.js`
   - `manifest.webmanifest`
   - `assets/`
   - `README.md`
3. تأكد أن ملف `index.html` موجود في جذر المشروع أو اضبط إعدادات النشر حسب مكانه.

مثال أوامر Git:

```bash
git init
git add .
git commit -m "Initial Ihsan Quran web app"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPOSITORY.git
git push -u origin main
```

## النشر على Vercel

المشروع Static Frontend، لذلك لا يحتاج Build Command.

1. ادخل على Vercel.
2. اختر **New Project**.
3. اختر Repository الخاص بالمشروع من GitHub.
4. إذا كانت الملفات في جذر المستودع، اترك Root Directory كما هو.
5. إذا كانت الملفات داخل مجلد مثل `ihsan`، اجعل Root Directory هو اسم هذا المجلد.
6. Framework Preset: اختر **Other** أو اتركه كما يحدده Vercel.
7. Build Command: اتركه فارغًا.
8. Output Directory: اتركه فارغًا أو اجعله `.` إذا طُلب منك.
9. اضغط Deploy.

بعد النشر، سيعمل الموقع عبر HTTPS، وهذا مهم لميزات مثل تحديد الموقع الجغرافي لمواقيت الصلاة.

---

## ملف vercel.json اختياري

يمكن إضافة ملف `vercel.json` في جذر المشروع إذا أردت إعدادات بسيطة للنشر:

```json
{
  "cleanUrls": true,
  "trailingSlash": false,
  "headers": [
    {
      "source": "/(.*)",
      "headers": [
        {
          "key": "X-Content-Type-Options",
          "value": "nosniff"
        }
      ]
    }
  ]
}
```

هذا الملف اختياري، والموقع يمكن أن يعمل بدونه.

## مصادر البيانات

يعتمد المشروع على مصادر خارجية لجلب بيانات القرآن، الصوت، الأذكار، ومواقيت الصلاة. لذلك يحتاج الموقع إلى اتصال بالإنترنت حتى تعمل كل الخصائص بشكل كامل.

> ملاحظة مهمة: نص القرآن الكريم لا يتم تعديله يدويًا داخل المشروع. يجب دائمًا الاعتماد على مصدر موثوق ومراجعة أي بيانات دينية قبل النشر العام.

## الخصوصية

- لا يوجد تسجيل دخول.
- لا يوجد Backend.
- لا يتم إرسال بيانات شخصية إلى قاعدة بيانات خاصة بالمشروع.
- المفضلة والإعدادات وآخر موضع قراءة يتم حفظهم محليًا في المتصفح باستخدام `localStorage`.
- ميزة مواقيت الصلاة قد تطلب إذن الموقع الجغرافي من المستخدم، ويتم استخدامه فقط لحساب مواقيت الصلاة.

---

## طريقة استخدام الموقع

1. من الصفحة الرئيسية اضغط **ابدأ القراءة** للانتقال إلى آخر موضع محفوظ أو فهرس السور.
2. اختر السورة التي تريد قراءتها.
3. استخدم وضع القراءة المتصلة أو قراءة البطاقات حسب تفضيلك.
4. اضغط على الآية في القراءة المتصلة لعرض التفسير أسفلها.
5. استخدم البحث للوصول إلى آية أو كلمة داخل القرآن.
6. افتح قسم الأذكار واختر الباب المناسب.
7. استخدم المسبحة الإلكترونية للتسبيح اليومي.
8. افتح مواقيت الصلاة لمعرفة الصلاة القادمة والوقت المتبقي عليها.
9. استخدم المفضلة لحفظ الآيات أو الأذكار المهمة.
